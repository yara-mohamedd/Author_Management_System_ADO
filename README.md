📚 Author Management System

A simple Console Application built with .NET for managing authors in a publishing company.
This project was created as part of my learning journey to practice database integration, CRUD operations, and building structured applications.

🚀 Project Overview

The system helps manage author records that were originally stored in Excel sheets and transforms them into a more organized and efficient database-driven solution.

The application supports managing data for 200+ authors through a clean and simple console interface.

✨ Features
➕ Add new authors
❌ Remove existing authors
📋 View all authors
🗄️ Database integration
🔄 CRUD operations
🧪 Sample data for testing
🛠️ Technologies Used
C#
.NET
SQL Server
ADO.NET 
Console Application
